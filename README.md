# Evaluacion de un sistema de recomendaciones en e-commerce Analisis A-B del embudo de conversion

Este proyecto analiza la efectividad de un nuevo sistema de recomendaciones en una tienda de comercio electrónico mediante un experimento A/B. El objetivo es determinar si dicho sistema mejora la conversión de usuarios en al menos un 10% en las distintas etapas del embudo de ventas: desde la visualización del producto hasta la compra. Se realiza un análisis exploratorio de los datos para garantizar su calidad y consistencia, se valida la correcta ejecución del experimento (incluyendo la duración, aleatorización y equilibrio entre grupos) y se utilizan pruebas estadísticas (prueba z) para comparar tasas de conversión entre los grupos de control y prueba. Al final del análisis, se presentan conclusiones basadas en la significancia estadística y recomendaciones claras sobre si debe implementarse o no el nuevo sistema de recomendaciones a gran escala.
